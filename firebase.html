¿Qué es Firebase?

Según su propia definición, Firebase es un conjunto de herramientas orientadas a la creación de aplicaciones de alta calidad, al crecimiento de los usuarios y a ganar más dinero

Todos los datos de Firebase Realtime Database se almacenan como objetos JSON. La base de datos puede conceptualizarse como un árbol JSON alojado en la nube. A diferencia de una base de datos de SQL, no hay tablas ni registros.

Firebase Realtime Database permite anidar datos en un máximo de 32 niveles

Sin embargo, cuando obtienes datos de una ubicación de la base de datos, también se recuperan todos los nodos secundarios.

Ventajas de Firebase

*Ganar Dinero con Firebase

*Multiplataforma

*Adiós WebServices (no necesitas api o servicios en la nube)

Desventajas de Firebase

*Pruebas en la nube limitadas, con la cuenta gratuita los accesos se vuelven limitados

*Precios, las cuentas con mejores herramientas de firebase tienen un costo alto

*Base de datos no tan complejas, al querer hacer una aplicacion robusta se tendra que contar con un mejor servidor para mantener la base de datos

*Límite de conexiones usuarios simultáneas, tiene un limite de usuarios a la hora de ingresar a la aplicacion asi que solo se podria realizar una aplicacion basica

¿Como mejora tu ambiente informatico?

- No necesitas un conocimiento extenso de bases de datos

- Te informa sobre los fallos que están ocurriendo en tu proyecto a la hora de desarrollarlo y compilarlo

- Soporta muchos lenguajes de programación, por lo que no necesitas aprender un nuevo de lenguaje que se adapte a firebase

- El entorno firebase brinda una buena optimizacion a las aplicaciones en desarrollo
